# iCalendar

.NET library for parsing, validating, and serializing iCalendar (text/calendar) data.

## Specification

Implements [RFC 5545](https://www.rfc-editor.org/rfc/rfc5545.html) - Internet Calendaring and Scheduling Core Object Specification.

See [specs.json](specs.json) for complete specification references.
